ConsoleForms

A C#.NET library allowing to make winform-like componets inside console windows.

View the Project on GitHub Joyesh/ConsoleForms

The easiest way to move from WinForms to console without losing much.

ConsoleForms is all about making almost every WinForm function into a .dll file, If you need to move from WinForms to Console, you can use this to almost effortlessly port your WinForm code, all you need is a few changes.

Why was this created?

My friend, @SirJosh3917 created his own dll for this once, but it was never finished & he can't find it anymore, so I decided to make my own.

Where can I get it?

Right now this project isn't complete, but if you're good at C#.NET, you can help us code it! Just get the latest development branch and start working!
